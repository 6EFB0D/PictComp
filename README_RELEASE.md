# PictComp - 画像一括圧縮アプリ

フォルダを選択するだけで、複数の画像を一括して圧縮・リサイズできるデスクトップアプリケーションです。

**v0.1.0 プレリリース** | 初回起動から14日間、全機能を無料でお試しいただけます。

---

## 主な機能

- **多形式対応**: JPEG、PNG、HEIC、WebP形式に対応
- **一括処理**: フォルダ内の画像を一度に処理
- **プリセット機能**: PowerPoint用、ブログ用、SNS用、Web用など用途別テンプレート
- **リサイズ機能**: 長辺ピクセル指定による自動リサイズ（アスペクト比維持）
- **プレビュー機能**: 圧縮前後の画像を比較表示
- **EXIF対応**: メタデータの保持/削除を選択可能（トライアル・Pro版）

---

## 必要な環境

- **Python 3.8以上**
- **Windows 10/11** または **macOS 10.14以上**

---

## インストール

### 1. ダウンロード

[リリースページ](https://github.com/6EFB0D/PictComp/releases) から最新版をダウンロードし、解凍してください。

### 2. 依存パッケージのインストール

```bash
cd PictComp
pip install -r requirements-gui.txt
```

（`requirements-gui.txt` が無い場合は `pip install Pillow pillow-heif` を実行）

---

## 起動方法

```bash
python gui_main.py
```

---

## 基本的な使い方

1. **入力フォルダ**と**出力フォルダ**を選択
2. プリセットを選ぶか、圧縮設定を調整
3. （任意）**プレビュー**ボタンで圧縮結果を確認
4. **圧縮開始**ボタンをクリック

---

## プリセット

| プリセット | 用途 |
|-----------|------|
| PowerPoint用 | スライド挿入向け（300KB目標、1920px） |
| ブログ用 | ブログ記事向け（200KB目標、1600px） |
| SNS用 | Instagram等向け（500KB目標、2048px） |
| Web用（高品質/軽量） | Webサイト向け（WebP形式） |
| メール添付用 | メール送信用（150KB目標） |
| PNG透過保持 | PNG形式で透過を保持 |

---

## 対応形式

| 入力 | 出力 |
|------|------|
| JPEG, PNG, HEIC, WebP | JPEG, PNG, WebP |

## トラブルシューティング

### HEIC形式が読み込めない場合

```bash
pip install pillow-heif
```

macOSの場合：
```bash
brew install libheif
```

### メモリ不足になる場合

一度に処理するファイル数を減らしてください。無料版は1回20枚までです。
