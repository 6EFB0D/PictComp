# PictComp ヘルプ

## 閲覧方法

### ブラウザで閲覧
`index.html` をブラウザで開いてください。

### Material for MkDocs でビルド
検索機能付きのドキュメントを作成するには、プロジェクトルートの [BUILD_MKDOCS.md](../BUILD_MKDOCS.md) を参照してください。

## 構成

| ファイル | 内容 |
|---------|------|
| index.html | はじめに |
| getting_started.html | 基本的な使い方 |
| compression.html | 圧縮・形式変換 |
| presets.html | プリセット |
| file_selection.html | ファイル選択とサムネイル |
| exif.html | 撮影情報（EXIF） |
| batch.html | 一括処理 |
| viewer.html | 画像ビューア |
| troubleshooting.html | トラブルシューティング |
| license.html | ライセンス・お問い合わせ |

上記HTMLはアプリに同梱され、「ヘルプを開く」で表示されます。Markdown（*.md）はMkDocs用のソースです。
