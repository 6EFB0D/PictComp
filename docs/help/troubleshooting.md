# トラブルシューティング

## HEIC形式が読み込めない

インストーラでインストールした場合、通常はそのまま利用できます。読み込めない場合は、お使いの環境でアンチウイルスソフトの除外設定を確認するか、お問い合わせください。

## メモリ不足

大量の画像を一度に処理するとメモリ不足になることがあります。ファイルを選択して処理するか、バッチに分けて実行してください。

## ログの確認

処理ログは以下の場所に保存されます。

- Windows: `C:\Users\<ユーザー名>\.pictcomp\logs\app.log`
- macOS/Linux: `~/.pictcomp/logs/app.log`

## 無料版の制限

無料版は1回20枚までです。Pro版（準備中）は無制限です。プレリリース期間中は14日間全機能を無料でお試しいただけます。
