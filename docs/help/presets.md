# プリセット

用途別の圧縮設定テンプレートです。

| プリセット | 用途 | EXIF |
|------------|------|------|
| PowerPoint用 | 300KB目標、1920px | 選択可 |
| ブログ用 | 200KB目標、1600px | 削除固定 |
| SNS用 | 500KB目標、2048px | 削除固定 |
| Web用（高品質/軽量） | WebP形式 | 削除固定 |
| メール添付用 | 150KB目標 | 削除固定 |
| PNG透過保持 | 透過を保持 | 削除固定 |
| アーカイブ用 | 1000KB目標 | 保持 |

## Web掲載用プリセットについて

ブログ用・SNS用・Web用・メール添付用・PNG透過保持は、Web掲載が想定される用途のため、**セキュリティ上の理由からEXIF（撮影情報）を引き継げない**ようになっています。GPS情報や撮影日時などが含まれると個人情報の漏洩リスクがあるため、これらのプリセットではEXIF削除が固定されています。
