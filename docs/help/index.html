はじめに

PictComp（ピクトコンプ）は、画像ファイルを一括で圧縮・リサイズ・形式変換できるデスクトップアプリケーションです。フォルダを選択するだけで、複数の画像を一度に処理できます。

主な機能

  • 多形式対応: JPEG、PNG、HEIC、WebP、TIFF、BMP形式に対応
  • 形式変換: 圧縮と同時に出力形式を変更可能（例: HEIC→JPEG、BMP→PNG）
  • 一括処理: フォルダ内の画像を一度に処理
  • プリセット: PowerPoint用、ブログ用、SNS用など用途別テンプレート
  • 撮影情報（EXIF）: メタデータの保持/削除、閲覧、ファイル日時・ファイル名の変更
  • プレビュー: 圧縮前後の比較表示

必要な環境

  • Windows 10/11 または macOS 10.14以上

起動方法

インストーラでインストールした場合、スタートメニューまたはデスクトップのショートカットから起動してください。Pythonやモジュールのインストールは不要です。
