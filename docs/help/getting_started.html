基本的な使い方

1. フォルダを選択

  1. 入力フォルダ: 圧縮したい画像が入っているフォルダを「参照」ボタンで選択
  2. 出力フォルダ: 圧縮後の画像を保存するフォルダを選択（入力と同じフォルダも可）

2. 設定を決める

プリセットを選ぶか、以下の項目を調整します。

  • 目標サイズ: 1枚あたりの目標ファイルサイズ（KB）
  • 最大サイズ: リサイズ時の長辺ピクセル数（0=リサイズなし）
  • JPEG品質: 20〜100（高いほど画質良好、ファイルサイズ増）
  • 出力形式: auto（入力に合わせる）、jpg、png、webp、tiff、bmp
  • 形式変換のみ: チェックすると圧縮・リサイズを行わず形式のみ変換

3. ファイルを選択（任意）

対象ファイル一覧で、処理したいファイルを選択します。未選択の場合はフォルダ内の全ファイルが対象です。

  • 一覧表示: ファイル名のリスト（Ctrl+クリックで複数選択）
  • サムネイル表示: 画像のサムネイルをクリックして選択
  • 全選択 / 選択解除: ボタンで一括操作

4. 圧縮を実行

「圧縮開始」ボタンをクリックします。処理が完了すると結果が表示されます。
